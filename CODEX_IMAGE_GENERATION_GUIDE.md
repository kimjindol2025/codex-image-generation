### 2️⃣ 이미지 생성 방법

#### 📍 방법 1: Python PIL 사용 (권장)
```python
from PIL import Image, ImageDraw

# 이미지 생성
img = Image.new('RGB', (1280, 720))
draw = ImageDraw.Draw(img)

# 그리기
draw.ellipse([100, 100, 200, 200], fill=(255, 0, 0))

# 저장
img.save('/tmp/output.png')
```

**장점:**
- ✅ 가볍고 빠름
- ✅ 외부 의존성 없음
- ✅ Python 기본 패키지

#### 📍 방법 2: ImageMagick 사용
```bash
convert -size 1280x720 \
  gradient:red-blue \
  -pointsize 40 \
  -annotate +100+100 "Hello" \
  output.png
```

**장점:**
- ✅ 복잡한 이미지 처리 가능
- ✅ 배치 작업 효율적

### Q1: "image_generation 설정이 안 적용됨"

