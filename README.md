# Codex Image Generation

Claude Code + Codex를 연결하여 무료로 이미지를 생성하는 완벽한 워크플로우

## 🎨 기능
- Python PIL을 사용한 가벼운 이미지 생성
- ImageMagick을 사용한 고급 이미지 처리
- 자동화 스크립트로 배치 작업 가능
- 비용: 0원 (Codex 구독 모델 활용)

## 📁 파일 구조
```
├── README.md                          # 이 파일
├── CODEX_IMAGE_GENERATION_GUIDE.md    # 완전한 가이드
├── codex_image_automation.sh          # 자동화 스크립트
├── generate_image.py                  # Python PIL 기본
└── generate_advanced_image.py         # Python PIL 고급
```

## 🚀 빠른 시작
```bash
# 방법 1: 자동화 스크립트
bash codex_image_automation.sh

# 방법 2: Python 직접 실행
python3 generate_image.py
```
